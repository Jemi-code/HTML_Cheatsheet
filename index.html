Introduction

The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS), which defines the look and layout of the content in the webpage and scripting languages such as JavaScript, which define behavior and content of webpages. Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document. HTML is not a programming language; it is a scripting language.


In late 1990, a physicist Tim Berners-Lee in an attempt for fellow CERN researchers to use and share documents online, developed HTML as well as the browser and server software. There have been 5 versions of HTML since 1990 – HTML, HTML 2, HTML 3, HTML 4 and the one mostly used by all browsers, HTML 5. HTML 5 is what is used on most browsers and thus will where most of our lessons will be focused on.


As expected, HTML 5 is an updated version of HTML 4, with allocations for media, such as videos and others.

Basic HTML tags

There are 142 HTML tags available. Do not be discouraged, you do not need to know all of them right now, even experts do not use all the tags.


We will start with the first 5 basic tags; I think these have been there since forever.


Html tags

This is the container for all the code you will write, it is written directly below Doctype html and every other code you write, even CSS and JavaScript are written in side of this. They also indicate the beginning and end of every webpage. It is written as, <html>Your code</html>


It also contains other tags that add functionality to the web page that are not specifically the body or the content of your webpage such as the title of the webpage, external font or styling links and tags, the function of the webpage, the icon that will display among others.


Body tags

This is the container for the body of the webpage. All items or code under this shows the actual content of the webpage that is what the owner or designer of the webpage wants to put across. It is written as, <body>Your body code</body>


Header tags

Before you write your essay, there has to be a heading. It normally has a bigger font than other elements in the essay. This goes the same for a webpage, it is written as, <h1>Heading</h1>


There are other kinds of headings in an essay and webpage, smaller headings. In HTML we can write them,

<h2>Heading</h2> <h3>Heading</h3> <h4>Heading</h4> <h5>Heading</h5> <h6>Heading</h6>

Paragraph tags

Just as your essay needs paragraphs, your web page needs some as well. It is written as, <p>Paragraph</p>


Break tag

This tag leaves a line break in the text. It allows the text to look spaced out and breathable. It is written as, <br>

The break tag is called as a self-closing tag. This simply means that it does not need a closing tag like most tags.

We can do something cool with this. So, take out your text editors and let’s do something cool.

<!DOCTYPE html>
<html>
<body>
<h1>Hello World</h1>
<p>Rain drops on roses and whiskers on kitten</p>
<p>Snowflakes that fall on my nose and eyelashes</p>
<br>
<p>Wild geese that fly with the moon on their wings</p>
<p>These are a few of my favorite things</p>
</body>
</html>

Open this in your browser and see yourself smile.

Strike-through tags

Horizontal rule tag

This tag is used to insert a horizontal rule or a thematic break in an HTML page to divide or separate document sections. It is written as,

</body>
</html>

Open this in your browser and see how different it is from the first one

Media tags

Enough with text for now, let’s add some media elements to our document.


Image tag

This tag adds images to the webpage. It is written as,

<img src = “link for image” alt = “name of picture”>

Src means source and you are to put in the source of your image, if it is on the internet, indicate the link. If it is on the machine and the image is in the same folder just write the name of the image along with the extension. An example would be,

<img src = “cats are cute.jpg”>

If the picture is in a folder inside the folder where your HTML file is then it would be,

<img src = “/images/cats are cute.jpg>

If your image is not even close your folder, open the picture in your browser and copy the link.

Alt is short for alternate text and is used to give an alternate text for the image just in case the image does not show up due to connectivity issues or a broken link. An example for the use of alternate text is,

<img src = “link of cat” alt = “a picture of Mr. snuffles having a bath”>

You can also determine the width or height of an image and it is written as,

<img src = “link of cat” alt = “a picture of Mr. snuffles having a bath” height = “200px” width = “300px”>

This gives the picture of Mr. snuffles a height of 200px and a width of 300px. You can also use percentages, em/ rem or vh.

Video

There are two main ways to put a video into your webpage. They are,

  • <video autoplay muted loop controls poster = “url” preload = “auto or metadata or none”>
    <source src = “video-url” type = “video/mp4”>
    Your browser does not support HTML5 video </video>

    It looks scary, but don’t be scared, let me simply explain it to you.


    A video tag can have one or more source with different video extensions (.ogg or .mp4) and the browser chooses the one it will play.
    Autoplay specifies that the video will start playing as soon as it is ready.
    Controls will give video controls to the inputted video such as play, pause etc
    Loop means that the video will never stop, it will start over every time it is finished.
    Muted simply means that will be no sound for the video
    Poster it gives an image to be shown while the video is downloading, or until the play button is clicked.
    Preload it specifies how the author thinks the video should be loaded when the page loads.
    Auto means the video will load or play alongside the webpage
    Metadata means the browser should only load the metadata when the page loads
    None means the browser should not load the video when the page loads.
    Note: preload is ignored if auto play is present.
    “Your browser does not support HTML5 video” will display if your browser does not support HTML5 video.


  • Another way is using the iframe tag which us short for inline frame tag. It not only displays videos, but other documents or web pages inside it. An example is,

    <iframe width = "" src = "Video URL" frameborder = "0" allowfullscreen></iframe>

    Frameboarder allows you to alter the width of the border of the image.
    Allowfullscreen is a function that can help you view it as full screen.
